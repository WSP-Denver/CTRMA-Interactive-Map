SH 45SW Project

SH 45SW is an innovative four-lane toll road being built on state property between MoPac and FM 1626 that will bring relief to the rapidly growing area of northern Hays and southern Travis counties. This project balances environmental sensitivity with new infrastructure to manage congestion in the region.

  • Phase: Under Construction
  • Project Cost: $108 Million
  • Project Length: 3.6 miles
  • Opening Date: Spring 2019
  • Toll Rate: Approx. $1
  • Shared Use Path: 4.5 miles
  • Average Daily Traffic: TBD
  • More Information can be found at www.sh45sw.com
